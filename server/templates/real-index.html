Function

A website implements many cryptography functions.

Caesar cipher

It is a technique for replacing encryption. All letters in the plaintext are replaced backwards (or forward) by a fixed number and then replaced with ciphertext.

Keyword cipher

The Keyword Cipher uses a Keyword to rearrange the letters in the alphabet. These different letters are then substituted for the letters in the message to create a secret message.

Vigenere cipher

The Vigenere cipher is a method of encrypting alphabetic text by using a series of interwoven Caesar ciphers, based on the letters of a keyword. It is a form of polyalphabetic substitution.

Autokey plaintext

An autokey-plaintext cipher(also known as text-autokey) is a cipher that incorporates the plaintext into the key. It uses the previous message text to determine the next element in the keystream.

Playfair cipher

The Playfair cipher is a digraph substitution cipher. It employs a table where one letter of the alphabet is omitted, and the letters are arranged in a 5x5 grid(I takes place J).

Column permutation

The message is written out in rows of a fixed length, and then read out column by column in some scrambled order. Both the width of the rows and the permutation of the columns are defined by a keyword.

Double-Transposition

This is simply a columnar transposition applied twice. The same key can be used for both transpositions, or two different keys can be used.

RC4

RC4 is a stream encryption algorithm with a variable key length. It uses the same key for encryption and decryption, so it is also a symmetric encryption algorithm.

DES

The message is written out in rows of a fixed length, and then read out column by column in some scrambled order. Both the width of the rows and the permutation of the columns are defined by a keyword.

RSA

RSA is an asymmetric encryption algorithm. RSA is widely used in public key encryption and electronic commerce. The difficulty of factoring large integers determines the reliability of the RSA algorithm.

MD5

The MD5 message-digest algorithm is a widely used hash function producing a 128-bit hash value. It can still be used as a checksum to verify data integrity, but only against unintentional corruption.

DH key exchange

Diffie–Hellman key exchange(DH) is a method of securely exchanging cryptographic keys over a public channel and was one of the first public-key protocols within the field of cryptography.

Single machine practice

Single machine implements encryption and decryption practice

The client uses different cryptographic algorithms to encrypt the plaintext information, decrypt the ciphertext information and implement DH key exchange.
The cryptographic algorithm includes: Caesar cipher, Keyword cipher, Vigenere cipher, Autokey plaintext cipher, Playfair cipher, Column permutation cipher, Double-Transposition cipher, RC4, DES and RSA.
